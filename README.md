# Nguyễn Thị Mai Lan - Backend

## Hướng dẫn sử dụng

### Cài đặt môi trường

```
1. Cài đặt Nodejs
2. Cài đặt bun
```

### Cài đặt dự án

Chạy các lệnh bên dưới:

```bash
git clone https://github.com/Lanng21/food-BE.git
cd Order_Food_Backend
bun install
```

thay đổi tên file .env.example thành .env, sau đó chạy tiếp lệnh sau:

```bash
bun run dev
```
